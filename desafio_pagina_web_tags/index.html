Vamos ver algumas tags?

Esse artigo irá mostrar a funcionalidade de algumas das principais tags utilizadas no HTML que poderão te ajudar na hora de construir um site, então vamos lá aprender mais sobre

HTML

Tag - <h1 à 6> </&h1 à 6> - <hr>

Tag h útilizada para definirmos os titúlos e sub-titulos da nossa página por conveção se aconselha utilizar somente um h1 por página já os sub-titulos são conforme seu conteúdo necessite mais também não abuse para a página não ficar muito polúida.

Tag hr útilizado para traçar uma linha logo abaixo de seu texto.

Tag - <p></p> - <i></i> - <u></u> - <mark></mark> - <small></small> - <strong></strong> - <sup></sup> - <sub></sub>

Tag p útilizada para definir o paragrafo de seu texto.

Tag i transforma o texto em ítalico ja a Tag u deixa o texto sublinhado.

Tag u útilizada para deixar o texto menor que o texto padrão.

Tag mark útilizada para marcar o texto de amarelo.

Tag small útilizada para deixar o texto menor que o texto padrão.

Tag strong útilizada para deixar o texto em negrito.

Tag sup e subSup é útilizada para deixar o texto suspenso e Sub é útilizada para deixar o texto abaixo da linha.

Tag - <ol></ol> - <ul></ul> - <li></li> - <a></a>

Tag ol útilizada para criar uma lista ordenada.

Tag ul útilizada para criar uma lista não ordenada.

Tag li útilizada para inserir os items da lista.

Tag a útilizada para criar um link para o usuário.

Como esse exemplo:

Lista Ordenada

  1. <ol></ol>
  2. <ul></ul>
  3. <li></li>
  4. <a></a>
  5. <small></small>

Lista Não Ordenada

  • <ol></ol>
  • <ul></ul>
  • <li></li>
  • <a></a>
  • <small></small>

Tag - <font></font> - <del></del> - <abbr></abbr> - <blockquote></blockquote> -

Tag font essa tag deixou de se usada no html5 e foi substituida pela tag style que é útilizada para estilizar um texto por exemplo entre outros itens <font></font> <style><style> << esse modelo é usado no head do HTML já o <h1 style="font-size=24px"> é utilizado na tag especifica.

Tag del útilizada para marcar uma palavra, frase etc, como se não fosse mais utilizada.

Tag abbr útilizada para criar um balão de texto quando o usuário colocar o mouse em cima de alguma exemplo <abbr title="Folha de Estilo em Cascatas">CSS</abbr>.

Tag blockquote útilizada para inserir uma citação de um livro por exemplo em sua página ele deixa item posicionado de outra maneira na página mais distante da margem

O que é HTML?
Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.
